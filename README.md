# DriftMeet

DriftMeet is an 18+ random video chat app for meeting new people online through live video, audio, and text chat.

Website: https://driftmeet.app

## What DriftMeet Does

DriftMeet connects adults for random one-on-one conversations. Users can start a live chat, skip to another match, use text chat alongside video, and block or report a conversation when needed.

## Key Features

- 18+ random video chat
- Audio, video, and text chat
- Anonymous profile names during chat
- Google sign-in before matching
- Block and report controls
- Community Guidelines and moderation controls

## Safety

DriftMeet is built with a safety-first approach. The app uses sign-in, adult-only access, reporting, blocking, and moderation tools to reduce abuse and support a healthier random chat experience.
